一、我們如何收集您的個人信息

我們會在以下場景收集您的個人信息：

  • 註冊賬號時：姓名、手機號碼、電子郵箱
  • 購物下單時：收貨地址、收貨人信息、支付方式
  • 瀏覽商品時：瀏覽記錄、搜索關鍵詞（用於個性化推薦）
  • 客服溝通時：聊天記錄、問題反饋內容

二、我們如何使用您的個人信息

我們僅在合法、必要、透明的原則下使用您的信息：

  • 完成交易訂單、提供商品或服務
  • 發送訂單狀態通知、促銷活動信息（可隨時退訂）
  • 優化用戶體驗、改進產品功能
  • 保障平臺安全、預防欺詐行為

三、我們如何共享、轉讓、公開您的個人信息

我們不會向第三方洩露您的個人信息，除非：

  • 獲得您的明確授權同意
  • 根據法律法規要求或行政、司法機關要求
  • 為提供您需要的服務而委託第三方處理（如物流、支付機構）

四、您的權利

您可以通過以下方式管理您的個人信息：

  • 登錄賬號後修改個人資料
  • 聯繫客服申請刪除、更正個人信息
  • 關閉個性化推薦功能（路徑：設置→隱私→推薦管理）
  • 註銷賬號（需通過客服驗證身份）

五、我們如何保護您的個人信息

我們採取以下安全措施保護您的信息：

  • 數據加密存儲和傳輸（SSL/TLS協議）
  • 定期進行安全漏洞檢測和風險評估
  • 限制內部員工訪問權限
