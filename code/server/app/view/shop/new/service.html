1. 定義

  • 用戶：指註冊、登錄、使用本服務的自然人、法人或其他組織。
  • 本服務：指我們通過官網、客戶端、小程序等平臺向用戶提供的 [具體服務內容，如信息發佈、數據存儲、在線交易等] 及相關技術支持。

2. 用戶義務

賬號註冊：

  • 用戶需提供真實、準確的註冊信息，並及時更新個人資料。
  • 賬號所有權歸我們所有，用戶僅獲得有限的使用權，不得轉讓、出租、出借賬號。

合法使用：

  • 不得利用本服務從事違法犯罪活動（如詐騙、傳播淫穢信息、侵犯知識產權等）。
  • 不得干擾、破壞平臺正常運營（如惡意刷量、攻擊服務器、濫用漏洞等）。
  • 遵守平臺發佈的各類規則，不得發佈虛假、誤導性或違反公序良俗的內容。

保密義務：

  • 妥善保管賬號密碼，因用戶保管不善導致的損失，我們不承擔責任。
  • 不得洩露平臺未公開的技術信息、商業信息等。

3. 知識產權與內容授權

  • 我們在本服務中提供的內容（如軟件、商標、設計、數據等）知識產權歸我們或相關權利人所有。
  • 用戶在使用服務過程中發佈的內容（如文字、圖片、視頻等），視為授予我們全球範圍內、免費、非獨家、可再許可的使用權，用於平臺運營、推廣及服務優化。
  • 用戶需保證其發佈的內容不侵犯第三方知識產權或其他合法權益，否則需承擔全部法律責任。
  • 妥善保管個人賬號及密碼，因個人保管不善導致的賬號安全問題由會員自行承擔

4. 隱私保護

我們將按照《隱私政策》收集、使用和保護用戶個人信息，請您務必仔細閱讀並同意該政策。如您不同意，請勿使用本服務。

  • 因不可抗力（如地震、黑客攻擊、政府政策調整等）或第三方原因導致服務中斷或數據丟失，我們不承擔賠償責任。

責任限制：

  • 除法律另有規定外，我們對用戶因使用本服務產生的間接損失、附帶損失、可得利益損失等不承擔責任。
  • 若我們需承擔賠償責任，賠償總額不超過用戶因使用本服務而實際支付的費用（如有）。
