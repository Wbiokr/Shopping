1. 會員資格

年滿18週歲的自然人或合法註冊的法人實體均可申請成為本平臺會員。申請時需提供真實有效的個人信息，包括但不限於姓名、聯繫方式、身份證號碼（法人需提供營業執照等資質文件）。

2. 會員權利

會員享有以下權利：

  • 本網站會員資格取得方式：使用者進行本網站之會員註冊，即可取得會員資格。
  • 本網站會員權益須知：為維護本網站所有會員權益，於本網站使用會員服務之會員促銷優惠活動、購物金、贈與…等相關個人權益時，以會員本人綁定壹組會員帳號為限，個人資訊作為驗證會員身分之依據。若會員未通過本網站之會員身分驗證，本網站有權取消會員權益。
  • 當您申請入會時，即表示您已完整閱讀且同意本會員權益及顧客資料所載個人資料保護條款。
  • 本網站會員得享有網站上所有開放使用之優惠活動，包含享有新朋友註冊立即贈購物金優惠。
  • 會員同意本網站有權以網站公告方式不定期更新會員權益，且無條件同意網站更新內容，亦無需個別通知會員。

3. 會員義務

會員需遵守以下義務：

  • 遵守平臺各項規章制度及國家法律法規
  • 不得利用會員身份從事任何違法違規或損害平臺及其他會員權益的行為
  • 維護平臺良好秩序，不得發佈虛假信息、垃圾廣告或進行惡意評價
  • 妥善保管個人帳號及密碼，因個人保管不善導致的帳號安全問題由會員自行承擔

4. 運費計算標準

滿萬即免運，其餘一律宅配酌收120＄運費

5. 商品交貨時間及訂單處理規定

可下單即有現貨，下單後3天內寄出。

6. 發票處理規定

随貨寄

7. 商品退換貨及鑑賞期之規定說明

  • 依《消費者保護法》，消費者享有7天猶豫期之權益，收到商品超過7天，恕不接受退貨。
  • 務必完整保留包裹中的包裝材料，退貨商品須完整保護包裝，若回收商品後發現商品外觀有瑕疵，我們將保留退貨退款的權益。
  • 當您的退貨需求通過審核，我們會請宅配人員至您填寫的收件地址取回商品，請務必將發票一併放入退貨包裹中，將由我們負擔此次物流的費用。
  • 箬您退貨後訂單餘額不足滿額折扣優惠限制，原滿額折扣優惠將無法使用。已使用之優惠卷於完成結帳後即不再有效，退貨申請成立後也無法取回優惠卷；已使用之購物金，則會退回您帳戶
  • 退貨品項若為：組合品、同品項2、3、6入，不可拆分退貨

8. 免責聲明

因不可抗力、政府行政命令、黑客攻擊、系統故障等非平臺可控因素導致會員權益無法正常行使的，平臺不承擔賠償責任，但會盡力協助會員解決相關問題。

9. 爭議解決

如會員與平臺之間發生爭議，雙方應首先通過友好協商解決；協商不成的，任何一方均有權向平臺所在地有管轄權的人民法院提起訴訟。
